# 🌟 Ayesha’s Portfolio App

A modern and creative Flutter-based portfolio application designed to showcase my skills, projects, UI/UX work, and personal brand.  
The app provides a smooth, interactive experience with clean animations, professional layouts, and a minimal design theme.

---

## 🚀 Features

- 🎨 Modern UI/UX with clean design and smooth animations
- 📄 Download/View CV directly from the app
- 🖼️ Creative Designs Gallery (Graphic design, UI/UX, 3D work, etc.)
- 📱 Portfolio Items including app screens, case studies, and project previews
- 📬 Working Contact Form allowing users to send messages via email
- 🌗 Light/Dark Mode Support
- 🎞️ Video/Animations Showcase
- 🧭 Easy navigation with a structured and accessible layout
