## Built With
- Flutter (Dart)
- GetX (for state management & navigation)
- Dio (for downloading CV PDF)
- OpenFile (for opening PDF files in-app)
- carousel_slider (for automatic scrolling of projects showcase)
- url_launcher (to access URL)
- path_provider (to provide path for my CV)
- video_player (for displaying video tutorial of my projects)
- animated_background (for backround animation)
